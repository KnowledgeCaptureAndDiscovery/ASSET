The Accelerating Scientific workflowS using EarthCube Technologies (ASSET) project is developing a framework to capture scientific workflows in different domains of geosciences, characterize the type of work and the amount of effort involved in each activity, and map workflow activities to available tools and infrastructure. The ASSET workflow sketching interface enables scientists to specify their workflows and describe the steps involved.

More information about ASSET can be found in the main project page: https://www.asset-project.info/
